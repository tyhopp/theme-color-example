Broken because of render-blocking CSS

Shows how browsers behave when the user's preferred color scheme is dark, and there is enough render-blocking CSS that the underlays and scroll bars do not respect the dark color scheme preference.

To see it fixed by reducing render-blocking CSS, visit this page in a new incognito window.

To see it fixed using the <meta name="theme-color" content="..." /> tag, visit this page in a new incognito window.
